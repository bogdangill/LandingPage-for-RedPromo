# test task (build a landing page) for RED PROMO web-agency

state 30.09.20

Тестовое задание при трудоустройстве. Лендинг сам по себе простой, одностраничный(несмотря на то, что контент в нем подразумевает другие рабочие страницы), форма регистрации/отзывов отсутствует. Макет для лендинга включал в себя три версии: мобильники, планшеты и десктопы. Требовалось использование Bootstrap 4. Я сверстал данный макет, используя bootstrap 4.5, стартерпак от бутстраповского шаблона(типа popper.js, jquery min, etc для корректной работы слайдера и анимаций). Также использовал бутстраповскую сетку на флексах. Лендинг был сделан за 4 дня, т.к. пришлось вспоминать сам фреймворк - до этого был единичный опыт пользования. В данной версии необходимо пофиксить навигационную панель юзера в шапке для маленьких мобильников типа iPhone 5/SE для корректного отображения. Также нужно сделать нормальный прелоадер а не его имитацию. jQuery как раз поможет в этой задаче.

P.S. Сам лендинг AWD, не RWD. Отзывчивости нет, только адаптив. В ближайшем будущем залью еще один лендинг, где будет преимущественно респонсив с небольшой долей адаптива + BEM и слайдер своими руками на pure JS.
